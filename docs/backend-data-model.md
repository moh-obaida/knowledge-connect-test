# نموذج البيانات الخلفي

وصلة المعرفة تستخدم Firebase Realtime Database لحالة الغرف المباشرة، مع حفظ القوالب والنتائج المحلية في `localStorage`.

## الغرفة

- `schemaVersion`: إصدار نموذج البيانات الحالي.
- `boardVersion`: إصدار لوحة الحروف الحالية `arabic-5x5-alif-v2`.
- `roomCode`: رمز الانضمام.
- `status`: حالة دورة الحياة: `setup`، `active`، `paused`، `ended`، `archived`.
- `board`: خلايا اللوحة.
- `eventLog`: سجل أحداث مختصر للعب.
- `questionHistory`: سجل أسئلة وأحداث الجولة.
- `featureFlags`: مفاتيح جاهزية مستقبلية للحسابات والذكاء الاصطناعي والتكاملات.

## اللوحة

اللوحة الافتراضية ٥×٥، ٢٥ خلية، أول حرف `ا`. تبقى غرف `أ` القديمة قابلة للعرض عبر طبقة التطبيع دون تعديل صامت للبيانات.
