# خطة الحسابات المستقبلية

الحسابات المستقبلية اختيارية ولا يجب أن تمنع إنشاء لعبة سريعة أو انضمام الطلاب.

## الأدوار

- `guest`: معلم أو طالب بدون حساب.
- `teacher`: حساب معلم لحفظ البنوك والقوالب.
- `student`: طالب اختياري مستقبلاً.
- `admin`: إدارة المنصة.

## قواعد المنتج

- لا يوجد جدار تسجيل دخول.
- الطلاب لا يحتاجون حسابات.
- المعلم يستطيع المتابعة بدون حساب.
- أي حفظ سحابي لاحق يجب أن يكون اختيارياً وواضحاً.
