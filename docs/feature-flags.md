# مفاتيح الميزات

مفاتيح الميزات موجودة داخل `featureFlags` في الغرفة لتجهيز المنصة للتوسع دون فرض ميزات غير جاهزة.

## المفاتيح الحالية

- `authReady`: جاهزية الحسابات المستقبلية.
- `aiReady`: جاهزية توليد الأسئلة مستقبلاً.
- `integrationsReady`: جاهزية التكاملات.
- `powerUpsEnabled`: تفعيل البطاقات المساعدة.
- `practiceModeEnabled`: تفعيل وضع التدريب.

أي ميزة غير مفعلة يجب أن تعرض رسالة عربية واضحة ولا تمنع اللعب كضيف.
