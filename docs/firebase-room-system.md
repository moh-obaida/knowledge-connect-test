# نظام غرف Firebase

كل غرفة تحفظ في المسار `rooms/{roomCode}`. القراءة تمر عبر `normalizeRoomState` حتى تتحمل الغرف القديمة والحقول الناقصة.

## قواعد مهمة

- إنشاء غرفة جديدة يستخدم `defaultRoomState`.
- تحديث الغرفة يضيف `updatedAt` ويزامن `status` مع حالة اللعب.
- الاشتراك المباشر يعيد حالة مطبعة للواجهة فقط.
- لا يتم ترقية الغرف القديمة بصمت؛ تظهر رسالة للمعلم مع خيار `تحديث اللوحة`.

## اتجاهات الفوز

- الفريق الأزرق: من اليسار إلى اليمين.
- الفريق الأحمر: من الأعلى إلى الأسفل.
- مسارات الفوز المتعرجة محسوبة بجيران الخلايا السداسية.
