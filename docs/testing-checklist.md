# قائمة اختبار وصلة المعرفة

## أوامر إلزامية

- `pnpm check`
- `pnpm exec vitest run` عند وجود اختبارات
- `pnpm build`

## تدفق المتصفح

- الصفحة الرئيسية عربية وRTL.
- إنشاء غرفة جديدة لا يطلب حساباً.
- شاشة الطلاب لا تعرض أدوات المعلم.
- اللوحة ٥×٥ وبها ٢٥ خلية.
- أول خلية تعرض `ا`.
- اختيار حرف يعرض سؤالاً.
- كشف الإجابة لا يظهر للطلاب إلا عند ضغط المعلم.
- منح الخانة للفريق يحدث النقاط واللوحة.
- الأزرق يفوز من اليسار إلى اليمين.
- الأحمر يفوز من الأعلى إلى الأسفل.
- تقرير نهاية اللعبة محفوظ محلياً.

## الغرف القديمة

- `أ` القديمة تظهر كـ `ا`.
- لا يحدث تعديل صامت في Firebase.
- خيار التحديث يظهر للمعلم فقط عندما تكون الترقية آمنة.
